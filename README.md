# Agro IoT

A simple HTML website showcasing the Agro IoT project - smart agriculture solutions using Internet of Things technology.

## Project Overview

This website provides public information about our IoT-based agricultural monitoring and automation system. It includes features for real-time data collection, automated irrigation, and data analytics to optimize farming practices.

## Features

- Responsive design
- Information about IoT in agriculture
- Contact details
- Interactive elements

## How to View

Simply open `index.html` in any modern web browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
